# Control Flow

Control flow commands determine which lines of your scene execute, in what order, and how the engine moves between scenes. This section covers three areas:

| Page | Description |
|---|---|
| [conditionals.md](conditionals.md) | `*if`, `*else`, `*elseif` — execute blocks based on variable state or expressions |
| [navigation.md](navigation.md) | `*label`, `*goto`, `*goto_scene`, `*finish` — jump within and between scenes |
| [subroutines.md](subroutines.md) | `*gosub`, `*gosub_scene`, `*return` — reusable labeled blocks that return to the caller |
